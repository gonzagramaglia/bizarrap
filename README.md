### ⚡ Overview

This is a high-performance web application dedicated to showcasing the complete collection of Bizarrap Music Sessions. Designed for fans and lyric enthusiasts, it provides a distraction-free, premium environment where every session is paired with its official lyrics and deep search capabilities.

### ✨ Key Features

- **🌐 Complete Collection**: Access all 66 Bizarrap Music Sessions in one centralized hub.
- **🎤 Bar-by-Bar Search**: Revolutionary search engine that lets you find sessions by artist, number, or even specific rhymes/lines of lyrics.
- **🔗 Deep Linking**: Every session has its own unique URL (e.g., `/53` for Shakira, `/52` for Quevedo) for easy sharing.
- **🌗 Aesthetic Dark Mode**: A UI optimized for the neon-and-black aesthetic of the BZRP studio, featuring smooth glassmorphism effects.
- **🌍 Multilingual**: Native support for English, Spanish, and Portuguese.
- **📱 Responsive & Fluid**: Built with modern CSS (Tailwind 4.0) to ensure a flawless experience across mobile, tablet, and desktop.
- 💨 **Ultra Fast**: Optimized with Vite and React 19 for instantaneous navigation and zero-lag interactions.

### 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4.0 (Alpha)](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)
- **State Management**: React Context API
- **Fonts**: [Geist](https://vercel.com/font)
